class slgnn.data_processing.deepchem_datasets.DeepchemDataset(root, name, transform=None, pre_transform=None, pre_filter=None)¶

Bases: torch_geometric.data.in_memory_dataset.InMemoryDataset

The base class to convert Deepchem dataset formatted data to pytorch-geometric Dataset.

Parameters

  • root (str) – path to the dataset directory. (The parent directory of raw and processed directories.)

  • name (str) – name of the csv file without extension name.

  • transform – optional. The transformer transforms data on the fly.

  • pre_transform – optional. The transformer transforms data while converting raw data to graphs.

  • pre_filter – optional. The filter function filters data while converting raw data to graphs.

raw_file_names¶

name of the raw file. If the file is not found in raw directory, download method will be called.

processed_file_names¶

filename of processed data. If the file is not found in processed directory, process method will be called.

download()¶

Get raw data and save to raw directory.

process(verbose=0)¶

The method converting SMILES and labels to graphs.

Parameters

Fix the test set based on the given ratio. The test set has the same data distribution with the training set, especially when the dataset labels are very imbalanced.

Parameters

  • dataset (torch_geometric.data.DataSet) – the dataset to be used in training.

  • ratio (list) – list of floats indicating the splitting ratios. The numbers represent fractions of training, validating, and testing sets, respectively.

  • shuffle (bool) – Whether shuffle the dataset.

  • batch_size (int) – size of every mini batches.
